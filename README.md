🐘 Projet PHP
📝 Présentation
Ce projet est une application web développée en PHP natif. Il a pour but de démontrer la maîtrise des bases du développement backend, notamment la gestion des requêtes HTTP, le traitement de formulaires et l'interaction avec une base de données.

🛠 Installation et Lancement
Cloner le projet :

Bash
git clone https://github.com/JonathanRAGE/projet_php.git
cd projet_php
Configuration :

Importez le fichier .sql fourni (s'il existe) dans votre gestionnaire de base de données (ex: phpMyAdmin).

Configurez vos identifiants de connexion dans le fichier de configuration (souvent config.php ou db.php).

Exécuter :

Placez le dossier dans votre répertoire serveur (ex: www/ pour WAMP ou htdocs/ pour XAMPP).

Accédez à l'application via http://localhost/projet_php.

💡 Exemples d'utilisation
Gestion dynamique : L'application permet d'afficher et de modifier des données stockées en base de données.

Formulaires : Envoi et validation de données utilisateur avec traitement côté serveur.

Sessions : Gestion de la connexion utilisateur et protection de certaines pages via session_start().

Stack : PHP, MySQL, HTML/CSS.
